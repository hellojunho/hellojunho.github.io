```markdown
BubbleSort(Array arr of N element)
  for i=1 to N-1
      for j=1 to N-i
          if arr[j] > arr[i+1]
              then interchange(arr[j], arr[j+1])
          end if
      end for
  endfor
```

```markdown
    n = m
    m = temp
```

```markdown
Merge(A, p, q, r)
    $n_1$ = q - p + 1
    $n_2$ = r - q
    let L[1...$n_1$+1] and R[1..$n_2$+1] be new arrays

    for i = 1 to $n_1$
        L[i] = A[p + i -1]
    for j = 1 to $n_2$
        R[j] = A[q + j]

    L[$n_1$ + 1] = $\infty$
    R[$n_2$ + 1] = $\infty$

    i = 1
    j = 1

    for k = p to r
        if L[i] <= R[j]
            A[k] = L[i]
            i += 1
        else A[k] = R[j]
            j += 1
```

```markdown
Partition(A, p, r)
    x = A[r]
    i = p - 1

    for j = p to r - 1
        if A[j] <= x
            i += 1
            exchange A[i] with A[j]
    exchange A[i + 1] with A[r]
    return i + 1
```
